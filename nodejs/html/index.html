Tabela de comandos para criar um projeto em node express.
Comando Efeito
npm install express --save Faz o download de módulos em uma pasta no projeto em node.js express .
npm install nodemon -g Instala globalmente o nodemon na aplicação atualizando-a em cada alteração do código.
nodemon nomeDaAplicação Executa a aplicação na porta em que o servidor foi configurado no código.
